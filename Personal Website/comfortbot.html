ComfortBot Discord Bot

ComfortBot is a WIP Discord Bot I'm currently developing. As such, its functionality is limited right now; however, I've attached instructions and a view into its testing server so anyone can try out its easy-to-display commands and features.

Testing Instructions

1. Because ComfortBot is still in early development, I'm not keeping it online constantly. Please go to its Replit link and click "Run" at the top of the page in order to bring the bot online. The bot will probably take about a minute to come online fully, so please be patient! (You can watch Replit's terminal to see when it's done)

2. ComfortBot should now be online, and the viewport on the right should allow you to login to Discord or simply give yourself a nickname.

3. You should be able to test any available commands ComfortBot has been given. Note that because of the limited viewport, audio commands aren't truly testable, but any text command should be visibly functional. To get started, type "$help" to get a list of ComfortBot's current commands.
